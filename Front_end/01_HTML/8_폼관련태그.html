폼 관련 태그

form 태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그이다.
form 태그 내의 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 한다.
* action 속성 : form에 입력된 값들을 전송받을 서버의 클래스명 입력
* method 속성 : get/post 방식으로 전송 방식 지정
- get 방식 : url 주소에 form 태그를 통해 넘어가는 데이터가 보임
- post 방식 : url 주소에 form 태그를 통해 넘어가는 데이터 보이지 않음

submit 버튼을 눌렀을 때 action에 지정한 경로로 method에서 지정한 방식에 따라 input 태그에 입력한 값들을 전달한다.

필드셋의 제목을 작성하는 부분이다.


fieldset으로 묶은 영역별로 구분된다.
