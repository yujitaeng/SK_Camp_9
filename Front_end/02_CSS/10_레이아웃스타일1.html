>> display: inline 테스트

블럭 요소를 인라인 블럭 요소로 변경

>> display: inline-block 테스트
영역은 인라인요소로 변경되지만 내용은 블럭요소가 되어 width와 height 속성을 설정할 수 있다. (요소 간 작은 간격이 존재)
첫 번째 영역
두 번째 영역
세 번째 영역
네 번째 영역
다섯 번째 영역

인라인 요소를 블럭 요소로 변경

>> display: block 테스트
인라인 요소를 블럭 요소로 변경하면 width와 height값을 지정할 수 있고 세로로 쌓인다.
