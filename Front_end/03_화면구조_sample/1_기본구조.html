영역 관련 속성

1) content : 내용이 표시되는 영역
2) border : 요소의 테두리 영역
3) padding : content와 border 사이의 영역
4) margin : 타 요소와의 간격을 지정하는 영역
현재 div의 테두리를 포함한 가로 길이
= width 값 + 양쪽 padding 값+ 양쪽 border 값 + 양쪽 margin값
= 300px + (20px * 2) + (20px * 2) + (20px * 2)
= 420px
현재 div의 테두리를 포함한 세로 길이
= height 값 + 상하 padding 값 + 상하 border 값 + 상하 margin값
= 200px + (20px * 2) + (20px * 2) + (20px * 2)
= 320px
